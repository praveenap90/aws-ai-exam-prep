## 🚀 Deploy Your Own

### Option 1: GitHub Pages (Recommended)

1. **Fork this repository**
2. Go to **Settings → Pages**
3. Set **Source** to `Deploy from a branch`
4. Select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live at:
   ```
   https://YOUR_USERNAME.github.io/aws-ai-practitioner-portfolio
   ```

### Option 2: Clone & Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/aws-ai-practitioner-portfolio.git
cd aws-ai-practitioner-portfolio
# Open index.html in your browser — no build step needed!
open index.html
```

## 🔑 Key Exam Tips

- **Pass score:** 700/1000 (scaled scoring)
- **Exam length:** 85 questions, 90 minutes
- **Format:** Multiple choice & multiple response
- **Focus on:** Amazon Bedrock features (28% of exam!)
- **No hands-on labs** — it's a foundational/conceptual exam
